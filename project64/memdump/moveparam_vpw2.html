VPW2 moveset and param dumper helper

(new version that works with combo and shootfighting styles)

Usage

  1. make sure memdump.js is running in project64, or this won't do shit.
  2. start a match with whatever wrestlers you want to dump the data for.
  3. click the below link(s). use "Save Link As" or similar if you want to rename the file.
